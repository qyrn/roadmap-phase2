* Learned:
 * Stacked queries en PostgreSQL : `;` pour enchaîner deux requêtes SQL, différent de `||` qui est juste de la concaténation de chaînes
- Pattern d'extraction blind time-based : `CASE WHEN (SUBSTRING(password, N, 1) = 'X') THEN pg_sleep(10) ELSE pg_sleep(0) END`, brute-force position par position via Intruder trié par response time
- XML numeric character references (`&#x55;` = U, `&#x53;` = S) pour bypass un WAF qui filtre par pattern matching sur le body brut, alors que le parser XML décode avant d'envoyer au SQL
- Décalage entre couche de validation (WAF) et couche d'exécution (parser + SQL) = classe de vulnérabilité exploitable
- Signatures sqlmap côté logs : User-Agent par défaut, chaînes aléatoires 4 chars (`pYRp`), concaténations CHR() longues, volume anormal, 500 errors répétées
- Méthodo : face à un WAF, comparer la dernière payload qui marchait à celle qui échoue et tester chaque différence isolément, plutôt que trial-and-error
